新长征的路上

博客搭建并使用也有一段时间了，也遇到了一些问题，在此记录下，持续更新…

  1. 主页下浏览第二页时会显示页面不存在，并跳转到豆瓣音乐->

    原因是之前添加了豆瓣音乐的外链，可能此外链已失效导致。

  1. git push -u origin master到oschina时报错：Access denied exec request failed on channel 0 fatal: Could not read from remote repository.->

    重新生成并添加公钥后解决此问题。

  1. git fetch报错：fatal: unable to access ‘http://git.oschina.net/wbeam/Hexo_blog.git/‘: Unkown SSL protocal error in connection to git.oschina.net:443->
    设置代理：
    git config --global http.proxy http://172.17.71.126:8008
    查看代理：
    git config --global --get http.proxy

潍坊懿轩庄园

发表于 2016-05-27   |   分类于 景点   |  

懿轩庄园成立于2007年，起因为找个世外桃园，喝茶，品树，谈论天下大事，广交天下好友。刚开始的规模为20多亩农地，种植美国红枫等名树，还有桃园，杏园，菜园，鸡棚等，为周围朋友提供了一个远离都市喧嚣的休闲之地，也能品尝到绿色无公害的蔬菜和水果。

懿轩的发展
目前改善生态环境、提高人居质量，正成为我国城市建设的主旋律，为了解决空气污染、噪音、热岛效应等不利于人们身体健康的“城市病”，我国许多大中城市正致力于发展城乡一体的城市绿化，竞相为城市营造一道绿色的“生态屏障”，园林产业的发展也被人们所看好。懿轩庄园抓住发展的良机，扩大了庄园的规模，引进了更多品种的园林，如红玉兰，黄玉兰，黄金槐，海棠等树木，为小区兴建，道路园林提供了很好的选择。庄园也会适时根据市场需求，引进各种树木，花卉等，为城市发展和市民生活提供新的一块风景地。同时懿轩庄园也承接各类大型团体活动，可以来庄园垂钓，聚餐，K歌，打牌，搓麻等一系列的活动，为你，我，他构建一个交流沟通的平台。


为弘扬传承中华民族“孝老敬亲”的传统美德，响应潍坊市文明办关于开展“孝德文化”公益活动的通知，2015年重阳节前由潍坊好人网联合中国画都展览馆、懿轩庄园举办的—-“中国画都展览馆创作基地挂牌暨潍坊好人网第三届‘千寿宴’‘百寿图’展览仪式”圆满成功。
